### vim comman
i = insert mode
v = visual mode
ESC = back to normal mode
d - delete
dd - delete a line
Y - yank(copy)
YY - yank a line
p - paste
u - undo
a - append
A - append from end of line
o - open line(under)
O - open line(upper)
H - move to the top ofo the screen
L - move to the bottom of the screen
